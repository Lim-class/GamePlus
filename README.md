# 🎮 GAME+ Pro Dashboard

**GAME+** è una piattaforma web dinamica e interattiva che raccoglie una suite completa di mini-giochi classici, arcade e di società, sviluppata con un focus sull'esperienza utente (UX) e su un design estetico moderno.

## 🚀 Panoramica
La dashboard funge da hub centrale da cui è possibile accedere rapidamente a vari giochi. Il progetto è stato progettato per essere leggero, veloce e reattivo.

### Caratteristiche Principali
* **Interfaccia Intuitiva:** Design moderno con uno stile curato.
* **Ricerca in Tempo Reale:** Filtra i giochi disponibili istantaneamente tramite la barra di ricerca.
* **Feedback Visivo:** Effetti dinamici, animazioni fluide e una struttura organizzata.
* **Multi-Piattaforma:** Ottimizzato per browser desktop e mobile.

---

## 🛠 Tecnologie Utilizzate
* **Frontend:** HTML5, CSS3 (con CSS Grid/Flexbox), Tailwind CSS (in alcuni moduli).
* **Logica:** JavaScript (Vanilla) per la gestione della logica di gioco e del DOM.
* **Asset:** Integrazione di icone emoji e Google Fonts (Roboto/Inter).
